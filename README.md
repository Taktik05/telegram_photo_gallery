Telegram Photo Gallery
Бот для создания веб-галерей из отправленных фотографий. Пользователь отправляет фото боту, получает ссылку на веб-страницу с галереей.
Требования
- Docker и Docker Compose (рекомендуется)
- Или Python 3.11+ и pip (для запуска без Docker)
Старт с Docker
1. Клонируйте репозиторий:
git clone https://github.com/Taktik05/telegram_photo_gallery.git
cd telegram_photo_gallery

2. Создайте файл .env с токеном вашего Telegram-бота:
BOT_TOKEN=ваш_токен

3. Создайте пустой файл galleries.json (необходимо для монтирования в Docker):
echo {} > galleries.json

4. Запустите проект:
docker-compose up -d --build

5. Проверьте логи:
docker logs telegram_photo_gallery
Вы должны увидеть:
Бот запущен. Ожидание сообщений...
Running on http://0.0.0.0:8080

6. Остановка:
docker-compose down

Запуск без Docker
1. Установите зависимости:
pip install -r requirements.txt

2. Создайте файл .env с токеном бота.

3. Запустите веб-сервер в одном терминале:
python web.py

4. Запустите бота в другом терминале:
python bot.py

Использование
1. В Telegram найдите своего бота, отправьте /start.
2. Отправляйте фотографии (можно несколько). После каждой бот подтверждает сохранение.
3. Когда закончите, отправьте /done.
4. Бот пришлёт ссылку вида http://localhost:8080/gallery/...
5. Откройте ссылку в браузере на том же компьютере, где запущен веб-сервер. Галерея отобразится.
Примечания
Фотографии сохраняются в папку photos/ на хосте.
Метаданные галерей хранятся в файле galleries.json.
При повторном запуске ранее созданные галереи остаются доступными.
Ссылка от бота может не быть кликабельной в Telegram из-за использования localhost. Скопируйте её вручную в браузер.
